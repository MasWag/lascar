Functor SetExt.Make

module Make (E : OrderedTypeExt.T) : S  with type elt = E.t
Functor building an implementation of the set structure given a totally ordered, stringable type.
Parameters:
E : OrderedTypeExt.T 
include Set.S
module Elt: OrderedTypeExt.T  with type t = elt
val map : (elt -> elt) -> t -> t
map f s applies function f to each element of set s
val power : t -> t list
power_set s computes the power set of s. Note: the result is returned as a list of sets because sets of sets cannot be defined easily in a functorial way..
val extract : t -> elt * t
Extract an element from a set, returning the element and the updated set. Raise Not_found if the set is empty
val to_string : t -> string
Returns a string representation
